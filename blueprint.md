# Lotto Number Generator

## Overview

This is a simple web application that generates 5 random lotto numbers.

## Features

*   Generate 5 unique random numbers between 1 and 45.
*   Displays the generated numbers to the user.
*   Modern and visually appealing design.
*   Responsive for mobile devices.
*   **Drawing Simulation:** Numbers appear one by one with a scale and rotate animation.
*   **Sorted Results:** Numbers are displayed in ascending order for better readability.

## Implemented Features & Design

*   **Initial Version:** Basic HTML, CSS, and JavaScript for generating and displaying 5 lotto numbers.
*   **Modern UI Update:** Applied a dark theme with vibrant purple accents and glowing effects.
*   **Web Components:** Refactored the lotto ball into a custom element `<lotto-number-display>` with its own Shadow DOM.
*   **Simulation & UX:** Added a 0.6s delay between drawing each number and disabled the button during the process to enhance the user experience.
