# Lotto Number Generator

A modern, visually appealing web application that generates unique lotto numbers (5 numbers between 1 and 45) using Web Components and modern CSS.

## Features

- **Modern UI:** Vibrant colors, glowing effects, and smooth animations.
- **Web Components:** Encapsulated logic and styles using Custom Elements and Shadow DOM.
- **Responsive Design:** Optimized for both mobile and desktop views.
- **Dynamic Styling:** Lotto balls are color-coded based on their number range.

## Tech Stack

- **HTML5** (Semantic structure)
- **CSS3** (Baseline features: Flexbox, Gradients, Shadows, Transitions)
- **JavaScript** (ES Modules, Web Components)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/naver-sangil/productbuilder-week1.git
   ```
2. Open `index.html` in your browser.

## Project Structure

- `index.html`: Entry point.
- `main.js`: Contains the logic and `lotto-number-display` Web Component.
- `style.css`: Global styles and layout.
- `blueprint.md`: Project roadmap and design decisions.
